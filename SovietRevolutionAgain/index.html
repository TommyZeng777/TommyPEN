    # 故事梗概

    • 体制内的清醒者：米哈伊尔·伊万诺维奇·沃罗宁作为苏联部长会议副主席，在1980年代末期逐渐意识到苏联体制的僵化和腐败问题，与总书记戈尔巴乔夫的改革路线产生分歧。

    • 顿悟时刻：沃罗宁在列宁格勒视察时目睹国营商店的腐败和物资短缺，与老布尔什维克尼古拉耶夫的交谈让他彻底认清了修正主义路线的危害。

    • 政治决断：回到莫斯科后，沃罗宁在政治局会议上公开批评现行政策，提出回归列宁主义原则的"新方向"计划，与戈尔巴乔夫彻底决裂。

    • 人民支持：沃罗宁的讲话通过地下电台传播后，获得基层党员和工人群众的热烈响应，一场自下而上的社会主义复兴运动开始形成。

    • 历史抉择：面对克格勃的监视和党内保守派的犹豫，沃罗宁决定前往乌拉尔工业区直接发动工人阶级，一场关乎苏联命运的斗争即将展开。
