Vue组件构成原创

# Vue 组件结构深度解析

在前端开发中，Vue 作为一款流行的渐进式 JavaScript 框架，其组件化开发模式极大地提升了代码的可维护性和复用性。本文将详细介绍 Vue 组件的基本构成，帮助你深入理解 Vue 组件的工作机制。

# Vue 页面基本组成部分

Vue 组件通常由三个主要部分组成，分别是：

# 1.template —— 结构（HTML）

这是你组件的“长相”部分，用来描述这个组件在页面上渲染成什么样子。

<template>
  <div class="my-component">
    <h1>{{ title }}</h1>
    <button @click="handleClick">点击我</button>
  </div>
</template>

# 2. script—— 逻辑（JavaScript）

这是组件的功能部分，用来定义数据、方法、生命周期等。

<script>
export default {
  name: 'MyComponent',
  data() {
    return {
      title: 'Hello Vue!'
    };
  },
  methods: {
    handleClick() {
      alert('按钮被点击了！');
    }
  }
};
</script>

# 3.style —— 样式（CSS）

这是组件的外观样式部分。你可以选择是否加上 scoped 来让样式只对当前组件生效。

<style scoped>
.my-component {
  background-color: #f0f0f0;
  padding: 20px;
  border-radius: 10px;
}
</style>

  <div class="my-component">
    <h1>{{ title }}</h1>

};
</script>
