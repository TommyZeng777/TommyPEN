VuePress 样式调调整探索

# 前言

不知道有没有同志和我一样，一直没搞懂这些主题的样式是怎么设置出来的，作为爱好者，每系统学习过，Vue 这些都不太会。不过经过我的初步研究得出了不少有价值的成果，希望对各位初学的同志的有帮助，毕竟很多人网页已经初具雏形，突然有个不喜欢的样式又改不了可能会很糟心。 但实事求是的看，遇到问题狂补前端知识也不现实。当然最稳妥的还是一边补知识一边修改自己的博客，一边倒的去做一遍的工作 ( 指改网页样式和学习新知识 ) 都是不现实的，一边学习一边实践效果最好。

警告

受制于水平限制，很多解释不可能都对，后续可能会翻新这些内容，希望各位同志能积极给我提意见。

# VuePress 组件样式实现方法

目前 Vue 实现样式主要有两种方法:

  • 主题文件夹内的各类vue文件以及style文件
  • .vuepress 文件内的 styles 文件夹内的 css/styl 文件

优先级: style > 主题内置

所以侧面也说明了 Vuepress 天生的高自定义程度，但确实需要一定的前端能力才能驾驭

# style的样式位置

Blog-File/
└── docs/
    └── .vuepress/
        └── styles/
            ├── index.styl (样式文件)
            └── 其他样式文件

在这个路径上，就可以找到 styl 样式文件， 样式代码和 CSS 代码差不多，大同小异

我的常用方法是在dev模式下，使用浏览器的DEVTOOLS查看元素，然后在Elements标签下找到对应的元素，然后复制到自己的styl文件中，然后修改样式，最后保存即可。
